# AgentBrain v2 Skill

Production-grade memory provider for Hermes Agent.

## Installation

```bash
hermes skill install agentbrain
```

## Usage

```bash
hermes config set memory.provider agentbrain
```

## Features

- Temporal knowledge graph
- Hierarchical memory tiers
- Native Obsidian export
- Automatic entity resolution
- Designed for long-running agents
